# README

Jump to recent files just like in IntelliJ IDEA editors.

## Features

Bindings:

```json
{
	"key": "ctrl+e",
	"command": "jump-to-recent.quickPick",
	"mac": "cmd+e"
},
{
	"key": "ctrl+shift+e",
	"command": "jump-to-recent.quickPickBack",
	"mac": "cmd+shift+e"
}
```
